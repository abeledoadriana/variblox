Tipos de datos

-

¿Qué son los tipos de datos?

Los tipos de datos son clasificaciones que indican la naturaleza de la información que una variable puede contener.

JavaScript reconoce varios tipos de datos fundamentales, cada uno con características específicas.

En JavaScript no es obligatorio declarar el tipo de dato de una variable cuando la creas (es un lenguaje de tipado dinámico).

Solo indicas que vas a crear una variable (let, const, var), pero no especificas si será un número, texto, booleano, etc.
El tipo se asigna automáticamente según el valor que le pongas, y puede cambiar en tiempo de ejecución.

Tipos de datos

Tipos de datos más comunes en JavaScript:

Number (Números)
 Para cualquier valor numérico, ya sean enteros o con decimales.

Ejemplo: 

let edad = 15;       // Número entero

let estatura = 1.75; // Número con decimales

String (Cadenas de Texto)
Para caracteres alfanuméricos. Texto, palabras, frases. Siempre van encerrados entre comillas simples (') o dobles (").

Ejemplo: 

let nombre = "Ana";

let saludoAna = '¡Bienvenidos!';

Boolean (Booleanos)
Representan valores lógicos: true (verdadero) o false (falso).

Son fundamentales para las decisiones.

Ejemplo: 

let llueve = true;

let haySol = false;

Array (Arreglos/Listas)
Son colecciones ordenadas de elementos. Por ejemplo en una lista de frutas. Se definen con corchetes [] y los elementos se separan por comas.

Ejemplo: 

 let tesorosEncontrados = ["manzana", "banana", "naranja", "pera"];

let numerosDeFrutas = [1, 7, 13, 20];

Object (Objetos)
Son colecciones de propiedades, donde cada propiedad tiene un nombre (clave) y un valor. Permiten representar entidades más complejas, como por ejemplo la descripción de un jugador.

Se definen con llaves { }.

Ejemplo: 

let jugador = {  

nombre: "Juan",  

edad: 14,  

rango: "Principiante",

tieneMedalla: true };
